Task Manager App
React Context API LocalStorage GitHub Netlify

A task management application built with React, using Context API for global state management and localStorage to persist tasks across browser sessions. Users can add, complete, and delete tasks with a clean, responsive interface — deployed on Netlify via GitHub.

What I built & learned
  • Built reusable React components with clean component architecture
  • Used Context API to manage global task state without prop drilling
  • Implemented localStorage persistence so tasks survive page refresh
  • Handled add, toggle-complete, and delete task operations
  • Deployed seamlessly from GitHub repository to Netlify
